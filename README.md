# Fishify-e-commerce
System documentation


##About
	Fishify is a seafood E - Commerce for shopping online, built with jetpack compose and Firebase.

- Features
- Online shopping
- Secure authentication
- Tracks orders
- Recommends items 
- Tracks sales
